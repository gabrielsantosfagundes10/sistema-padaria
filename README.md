# sistema-padaria

Sistema web desenvolvido para gerenciamento operacional e financeiro de padaria, com foco em organização, controle administrativo e acompanhamento diário das atividades.

O sistema foi desenvolvido para uso real em ambiente de produção, sendo utilizado por um cliente para gerenciamento de encomendas, contas, fluxo de caixa e controle financeiro.

## Funcionalidades

- Sistema de autenticação com login
- Dashboard administrativo
- Controle de encomendas
- Controle de contas de clientes
- Avisos e lembretes operacionais
- Fechamento de caixa
- Controle de gastos
- Relatórios de lucros mensais e semanais
- Monitoramento de contas a vencer
- Visualização de encomendas do dia e do dia seguinte
- Interface responsiva

- movimentações financeiras;

## Objetivo do projeto

O objetivo do projeto foi criar uma plataforma administrativa simples e eficiente para auxiliar no gerenciamento operacional e financeiro de uma padaria.

O sistema foi desenvolvido com foco em usabilidade, organização das informações e praticidade no uso diário.

## Projeto em produção

Sistema utilizado em ambiente real para gerenciamento operacional e financeiro de padaria.

## Observações

Dados sensíveis, credenciais e informações financeiras foram removidos da versão pública do repositório.
